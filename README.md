# 🎬 Netflix Data Analysis

## 📌 About
Exploratory Data Analysis (EDA) on the Netflix dataset to uncover trends in content type, genres, ratings, and country-wise distribution of titles.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Tool:** Jupyter Lab

## 📊 What I Analyzed
- Distribution of Movies vs TV Shows on Netflix
- Top genres and content categories
- Country-wise distribution of titles
- Ratings breakdown across content
- Yearly trend of content added to Netflix

## 🔍 Key Steps
1. **Data Collection** — Used Netflix titles dataset
2. **Data Cleaning** — Handled missing values, removed duplicates
3. **EDA** — Explored patterns and distributions
4. **Visualization** — Bar charts, heatmaps, and trend lines
